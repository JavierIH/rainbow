# Rainbow
GUI wrapper for python modules

## Installation

### System dependencies

```bash
# Install wxpython
sudo apt-get install python-wxgtk2.8
```

## Execute

* Include your modules here: ``rainbow/modules``
* Import and load your modules here: ``rainbow/modules/__init__.py``
* Launch Rainbow

```bash
python launcher.py
```
